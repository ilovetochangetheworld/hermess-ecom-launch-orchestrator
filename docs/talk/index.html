安装与启用
02 / 09

先把技能装好，再开始跑商品。

如果已经安装 SkillHub 商店，直接安装电商技能；如果没有，先按官方文档只安装 CLI。

安装完成后，客户只需要给一个商品和几条事实，Hermess Agent 会把模糊输入拆成一条可确认的业务流程。

1. 先检查 SkillHub 商店 若已安装，跳到第 3 步；若未安装，打开官方安装说明。
2. 未安装时：只安装 CLI 按 https://skillhub.cn/install/skillhub.md 操作，选择仅安装 SkillHub CLI。
3. 安装电商技能 在 SkillHub 中安装 hermess-ecom-launch-orchestrator 技能。
4. 启动流程 对 Agent 说：运行“从 0 开张：你的 AI 电商合伙人”，我选择“小鸭子泡澡玩具”。
安装口径：已有商店直接装技能，未安装只装 CLI
全链路流程
03 / 09

Agent 像一个轻量电商团队一样连续工作。

1

选品

判断品类、卖点、风险，决定是否继续。

2

定价

把成本、平台和毛利变成价格区间。

3

文案

生成绑定产品事实的小红书笔记。

4

图片 Prompt

输出封面、功能、场景图的 Prompt 和拍摄清单。

5

客服

FAQ 承接用户问题，识别购买意图。

6

复盘

把评论痛点回流成下一轮选品种子。

重点：不演示直接发布，只交付发布准备每一步都有可见产物

客户如何复用
08 / 09

换一个商品，也能按同一套流程再跑一遍。

输入商品事实商品名、成本、目标平台、核心卖点、已有图片或链接。
确认阶段结果选品、定价、文案、图片 Prompt、客服、复盘，每一步都能修改。
沉淀经营资产把聊天结果变成文案包、图片 Prompt 包、客服 FAQ 和复盘建议。
交付启动包生成一份可分享的商品经营启动包，团队可以继续执行。
复用换品再跑下一次只换商品和平台，流程、质量门禁和交付格式保持一致。
客户拿走的是方法和资产，不是一次性演示适用于商品测试、运营提效和团队培训
